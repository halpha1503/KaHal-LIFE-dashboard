# PCスペース常時表示ダッシュボード 要件定義

作成日: 2026-03-31
バージョン: v1.0

---

## 1. 概要

iPad Gen 6をPCスペース横に横置きで固定し、作業中に常時表示するダッシュボード。
HTML/JS/CSSで実装し、GitHub Pages（KaHal-LIFEリポジトリ `docs/`）でホスティングする。
iPadのSafariでフルスクリーン表示して運用する。

### 設計思想
- PCスペースでの作業に特化した「集中補助」ツール
- 時計・ブラウザ・通知系は不要（別デバイスで代替可能）
- Apple的なスマートさ・余白・滑らかさをデザインの基準とする
- グラデーション背景（Haikei等で生成）をプリセットとして設定可能

---

## 2. 機能一覧

### F1 - Today タスク表示

| 項目 | 内容 |
|------|------|
| データソース | GitHub Issues API（KaHal-LIFEリポジトリ） |
| 取得条件 | ラベル `Today` が付いているIssue |
| 表示内容 | タイトル・カテゴリラベル（dev / career / life / routine） |
| 操作 | タップでF4（Issue詳細ビュー）を開く |
| 更新 | 一定間隔で自動更新 |

### F2 - Next Up（カレンダー）

| 項目 | 内容 |
|------|------|
| データソース | Google Calendar API |
| 表示件数 | 直近2〜3件 |
| 表示内容 | イベント名・日時 |
| 更新 | 一定間隔で自動更新 |

### F3 - ポモドーロタイマー

| 項目 | 内容 |
|------|------|
| 方式 | ポモドーロテクニック（作業→短休憩→長休憩のサイクル） |
| デフォルト値 | 作業: 25分 / 短休憩: 5分 / 長休憩: 15分 / 長休憩までのセット数: 4 |
| カスタマイズ | 上記4値を設定画面から変更可能 |
| セッション記録 | 完了セット数（本日）・総作業時間（本日） |
| リセット | 日付が変わったら自動リセット |
| 保存 | localStorage |

### F4 - Issue 詳細ビュー

| 項目 | 内容 |
|------|------|
| 起動 | F1のタスクをタップ |
| 表示形式 | モーダルまたはドロワーで展開 |
| 表示内容 | Issue本文（作業分解テーブルを含む） |
| 操作 | チェックボックスのON/OFF |
| 同期 | なし（ローカル保存のみ） |
| 保存 | localStorage（Issue IDをキーに保存） |

### F5 - 背景カラー設定

| 項目 | 内容 |
|------|------|
| 形式 | グラデーション背景（CSS gradient） |
| プリセット | Haikei（haikei.app）等で生成したグラデーションを複数登録 |
| 選択方法 | 設定画面のサムネイル一覧から選択 |
| 保存 | localStorage |

### F6 - Spotify連携【発展】

| 項目 | 内容 |
|------|------|
| 接続方式 | Spotify Web API（OAuth） |
| 機能 | 再生中の曲表示・プレイリスト選択・再生コントロール |
| 実装時期 | Phase 2以降 |

---

## 3. 実装フェーズ

| フェーズ | 機能 | 主な技術 |
|---------|------|---------|
| Phase 1a | F3（タイマー）+ F5（背景設定）+ レイアウト | HTML / CSS / JS / localStorage |
| Phase 1b | F1（Today）+ F4（Issue詳細） | GitHub Issues API / Personal Access Token |
| Phase 1c | F2（Next Up） | Google Calendar API / OAuth |
| Phase 2 | F6（Spotify） | Spotify Web API / OAuth |

---

## 4. 技術構成

| 項目 | 選択 |
|------|------|
| 実装 | HTML / CSS / Vanilla JS（単一ファイル構成） |
| ホスティング | GitHub Pages（KaHal-LIFE `docs/` ディレクトリ） |
| 閲覧環境 | iPad Gen 6 / Safari フルスクリーン / 横置き |
| 外部API | GitHub Issues API / Google Calendar API / （Spotify Web API） |
| ローカル保存 | localStorage |
| 認証情報 | GitHub PAT / Google OAuth クライアントID（環境変数または設定画面入力） |

---

## 5. 非機能要件

- 時計は表示しない（別デバイスで代替可能）
- Webブラウザ機能は持たない
- LINEやDiscord等の通知系は表示しない
- オフライン時はキャッシュ済みデータを表示する（APIエラー時のフォールバック）
